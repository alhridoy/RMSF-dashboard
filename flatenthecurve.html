Model Architecture

The LSTM model is using a type of neural network called a Long Short-Term Memory (LSTM) network, which is well-suited for analyzing sequential data.

The LSTM model is then defined and compiled, with two layers of 64 nodes each and a single output node. The model is trained using the training data, with 100 epochs and a batch size of 72.
